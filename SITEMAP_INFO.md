# Информация о Sitemap.xml

## Описание

Sitemap.xml автоматически генерируется backend сервером и содержит:
- Главную страницу сайта
- Все страницы автомобилей из базы данных

## Доступность

Sitemap доступен по адресу: **https://avtozakaz74.ru/sitemap.xml**

## Технические детали

### Backend Endpoint
- **URL**: `/api/sitemap`
- **Метод**: GET
- **Формат ответа**: XML

### Генерация

1. Backend делает запрос к API che168 для получения списка автомобилей
2. Максимальное количество автомобилей в sitemap: **10,000**
3. Если автомобилей больше 10,000, в sitemap включаются первые 10,000
4. Sitemap обновляется при каждом запросе (с кешированием на 1 час)

### Структура URL в sitemap

```xml
<url>
  <loc>https://avtozakaz74.ru/</loc>
  <lastmod>2025-02-03</lastmod>
  <changefreq>daily</changefreq>
  <priority>1.0</priority>
</url>

<url>
  <loc>https://avtozakaz74.ru/car/{infoid}</loc>
  <lastmod>2025-02-03</lastmod>
  <changefreq>weekly</changefreq>
  <priority>0.8</priority>
</url>
```

## Настройка в поисковых системах

### Google Search Console
1. Зайти в раздел "Sitemaps"
2. Добавить URL: `https://avtozakaz74.ru/sitemap.xml`
3. Google автоматически начнет индексацию

### Яндекс.Вебмастер
1. Зайти в раздел "Индексирование" → "Файлы Sitemap"
2. Добавить URL: `https://avtozakaz74.ru/sitemap.xml`
3. Яндекс начнет индексацию страниц

## Производительность

- **Кеширование**: Sitemap кешируется на 1 час (Cache-Control: public, max-age=3600)
- **Время генерации**: Зависит от количества автомобилей (обычно 1-5 секунд)
- **Размер**: При 10,000 автомобилях размер sitemap ~500KB

## Ограничения

- Максимальное количество URL в одном sitemap: 50,000 (по стандарту)
- Максимальный размер файла: 50MB (по стандарту)
- Текущая реализация ограничена 10,000 автомобилями

## Улучшения в будущем

Если количество автомобилей превысит 10,000, можно реализовать:
1. **Sitemap Index**: Разделить на несколько sitemap файлов
2. **Инкрементальное обновление**: Обновлять только измененные страницы
3. **Статическая генерация**: Генерировать sitemap статически при обновлении базы

## Проверка

Проверить валидность sitemap можно через:
- https://www.xml-sitemaps.com/validate-xml-sitemap.html
- Google Search Console → Sitemaps → Проверка
