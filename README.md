# Avtozakaz74 - Веб-сайт автомобильных объявлений из Китая

Веб-приложение для поиска и просмотра автомобилей с китайского рынка через API che168.com

## 🚀 Технологии

### Frontend
- **React** 18.3.1 - UI библиотека
- **Vite** 6.0.3 - Сборщик и dev-сервер
- **Tailwind CSS** 3.4.17 - CSS фреймворк
- **API**: che168 (https://api-centr.ru/che168/)

## 📋 Возможности

- 🔍 Поиск автомобилей с расширенными фильтрами
- 💰 Автоматический расчет цен в рублях с учетом всех сборов
- 📊 Актуальные курсы валют
- 🖼️ Галерея фотографий автомобилей
- 📄 Детальная информация о каждом автомобиле
- 📱 Адаптивный дизайн

## 🛠️ Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка зависимостей

```bash
cd frontend
npm install
```

### Разработка

```bash
npm run dev
```

Приложение будет доступно на http://localhost:5173

### Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `frontend/dist/`

## 🔑 API

Проект использует API che168 для получения данных об автомобилях.

Документация API: https://api-centr.ru/che168/docs

### Основные эндпоинты:
- `POST /che168/search_car` - Поиск автомобилей
- `POST /che168/get_car_info` - Детальная информация
- `POST /che168/getAvailableFilters` - Доступные фильтры

## 📝 Фильтры поиска

- Бренд автомобиля
- Модель
- Год выпуска (диапазон)
- Цена в рублях (диапазон)
- Пробег (диапазон)
- Тип топлива
- Объем двигателя (диапазон)
- Мощность двигателя (диапазон)

## 💡 Особенности

- Цены уже рассчитаны с учетом таможенных сборов и утилизационного сбора
- Автоматическое обновление курсов валют из API
- Proxy через Vite для обхода CORS ограничений
- Оптимизированная загрузка изображений

## 🔒 Безопасность

⚠️ **Важно**: API токен хранится в `frontend/src/lib/api.js`. Не коммитьте его в публичные репозитории!

Для production окружения рекомендуется:
- Использовать переменные окружения для токена
- Настроить backend proxy для API запросов
- Добавить rate limiting
